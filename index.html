Estruturas De Dados

Código com implementações para a disciplina CIC-116319 Estruturas de Dados. Os arquivos são feitos para funcionar em ambiente Unix, com o compilador gcc.

Uso

Os arquivos são feitos para exemplificar certos conceitos, facilmente identificados pelo nome do artigo. O código segue o padrão ANSI, e pode ser compilado com o seguinte comando:

gcc -Wall -ansi codigo_fonte.c -o nome_executavel

Por exemplo: gcc -Wall -ansi ola_turma.c -o oi

Erros

Em caso de erro, lembre-se que a internet é sua amiga, basta procurar um pouco. Se não houver jeito, entre em contato com o responsável pelo projeto.
